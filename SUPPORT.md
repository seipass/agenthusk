# AgentHusk Support

AgentHusk is pre-1.0 OSS maintained on a best-effort basis.

## Usage questions

Before opening an issue:

1. Run `node src/cli.js help` or `npx agenthusk help`.
2. Start with `node src/cli.js demo` or `npx agenthusk demo`.
3. Read [README.md](README.md), especially the trust model and limits.
4. Use a copied tree or snapshot and synthetic examples when reproducing a problem.

Use the GitHub bug-report or feature-request form for public discussion. Do not attach real reports, secrets, or sensitive paths.

## Security issues

Do not open a public issue. Follow [SECURITY.md](SECURITY.md) and use GitHub private vulnerability reporting.
